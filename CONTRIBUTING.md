# Contributing to BetterHome

Thank you for contributing.

1. Fork the repository and create a focused branch.
2. Build and test the application on Windows.
3. Keep unrelated changes out of the pull request.
4. Describe the user-visible behavior and include screenshots for UI changes.
5. Do not commit generated build output, personal settings, or credentials.

By contributing, you agree that your changes will be licensed under the MIT License.
